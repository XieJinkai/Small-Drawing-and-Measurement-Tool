# 小型绘图测量工具（支持画线、画圆弧）

### 🎯 目标

为解决自动测量出现的问题以及部分特殊测量需求，添加手动功能实现几何判断。

### 🛠 技术要点

- QT/C++ 
- CMake

### 🔥实现要点

- 能够导入图片作为画布，并能进行缩放进行绘制。

- 具备绘制直线、圆弧功能，并标注出直线的长度、圆弧的半径、夹角的大小

### 🤖 开发方式

使用Trae进行开发，代码使用 AI 生成，但数据构造、验证与说明由我独立完成。
